# Улучшения для страницы тарифов

## Проблема
Текущий дизайн может восприниматься как "инфоциганство" из-за:
- Агрессивных "скидок" (72%)
- Ярких градиентов
- "BEST VALUE" с звездочками
- Расплывчатых обещаний

## Решение: Профессиональный образовательный стиль

### 1. УБРАТЬ агрессивные скидки

**Было:**
```
Premium Yearly: $99.99/год
Экономия $260 (72%)
```

**Стало:**
```
Premium Yearly: $99.99/год
Вместо $29.99/месяц × 12
```

### 2. Заменить градиенты на сдержанные цвета

**Было:**
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
```

**Стало:**
```css
background: #1976d2  /* Просто синий Material Design */
border-left: 4px solid #ffc107  /* Акцент */
```

### 3. Убрать "BEST VALUE", заменить на информацию

**Было:**
```
★ BEST VALUE
```

**Стало:**
```
Самый популярный выбор
или
Рекомендация преподавателей
```

### 4. Конкретизировать features

**Было:**
```
- Приоритетная поддержка
- Полный доступ ко всему контенту
```

**Стало:**
```
- Ответ на вопросы в течение 24 часов
- 15+ протоколов массажа с видео
- 50+ 3D моделей анатомии
- Практические тесты после каждого раздела
- Сертификат с уникальным номером
```

### 5. Добавить образовательный контекст

```
Профессиональное обучение массажу
- Методики от лицензированных специалистов
- Программа одобрена [орган сертификации, если есть]
- Более 500 студентов прошли обучение
```

### 6. Честная формулировка цен

**Free tier:**
```
Бесплатный доступ
- Ознакомление со структурой курса
- 3 бесплатных урока
- Без доступа к сертификации
```

**Basic:**
```
Базовый курс
$9.99/месяц
- 4 основных протокола массажа
- Базовая анатомия
- Сопровождение в Telegram
```

**Premium:**
```
Полный курс
$29.99/месяц или $99.99/год
- Весь контент платформы
- Индивидуальные консультации
- Сертификат об окончании
```

### 7. Убрать триальный блок с градиентом

**Было:**
```tsx
<Paper elevation={3} sx={{
  background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)',
  color: 'white'
}}>
  <StarIcon /> Попробуйте Premium бесплатно!
</Paper>
```

**Стало:**
```tsx
<Alert severity="info" icon={<InfoIcon />}>
  <AlertTitle>3 дня бесплатного доступа</AlertTitle>
  Попробуйте все возможности платформы без оплаты
</Alert>
```

## Визуальный стиль

### Цветовая схема (вместо градиентов)
```
Primary: #1976d2 (синий Material Design)
Secondary: #455a64 (темно-серый)
Success: #2e7d32 (приглушенный зеленый)
Accent: #ffc107 (желтый для важных элементов)
```

### Типографика
```
Заголовки:
- H3: "Профессиональное обучение массажу"
- H6: "Выберите подходящий формат обучения"

Вместо:
- H3: "Тарифные планы"
- H6: "Выберите подходящий план"
```

## Социальное доказательство

Добавить секцию:
```tsx
<Box sx={{ mt: 6, textAlign: 'center', bgcolor: 'grey.50', p: 4 }}>
  <Typography variant="h5" gutterBottom>
    Более 500 студентов доверяют нам
  </Typography>
  <Grid container spacing={2} sx={{ mt: 2 }}>
    <Grid item xs={4}>
      <Typography variant="h4">15+</Typography>
      <Typography variant="body2">Протоколов массажа</Typography>
    </Grid>
    <Grid item xs={4}>
      <Typography variant="h4">50+</Typography>
      <Typography variant="body2">3D моделей</Typography>
    </Grid>
    <Grid item xs={4}>
      <Typography variant="h4">500+</Typography>
      <Typography variant="body2">Выпускников</Typography>
    </Grid>
  </Grid>
</Box>
```

## Честность в формулировках

❌ Избегать:
- "Уникальная методика"
- "Секреты профессионалов"
- "Стань мастером за 30 дней"
- "Гарантированный результат"
- "Только сегодня"

✅ Использовать:
- "Проверенные методики"
- "Рекомендации практикующих массажистов"
- "Программа рассчитана на 3 месяца обучения"
- "Мы поможем вам освоить основы"
- "Постоянная цена"
